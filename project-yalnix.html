Yalnix Operating System Kernel

Rice University | Unix-like kernel implementation in C

COperating SystemsVirtual MemoryConcurrency

Overview

I designed and implemented core operating system mechanisms for a Unix-like teaching kernel, focusing on process lifecycle management, interrupt-driven control flow, and memory subsystem correctness.

What I Built

  • Process scheduling and context switching infrastructure.
  • System call interface and trap/interrupt handling path.
  • Virtual memory subsystem with custom page tables and TLB management.

Engineering Focus

  • Kernel correctness across multi-threaded and multi-process workloads.
  • Defensive debugging for race conditions and memory mapping edge cases.
  • Performance validation with profiling-oriented stress scenarios.

Outcome

The project delivered a stable kernel capable of handling process orchestration, memory translation, and user-kernel transitions with reliable behavior under concurrent workload execution.
