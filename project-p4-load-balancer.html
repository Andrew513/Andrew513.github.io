Programmable Network Load Balancing

Rice University | P4, ECMP, and flowlet switching

P4Flowlet SwitchingMininetScapy

Overview

This project focused on reducing packet reordering while preserving multipath utilization by implementing flowlet-aware forwarding in a programmable data plane.

What I Built

  • Implemented flowlet switching using P4 register arrays and inter-packet timing gap logic.
  • Built reproducible Mininet testbeds with heterogeneous path delays for realistic behavior validation.
  • Added dataplane-level monitoring through per-port counters and query-packet collection workflows.

Outcome

  • Reduced packet reordering compared with per-packet load balancing strategies.
  • Maintained multipath usage with improved forwarding stability under load.
  • Established repeatable test and telemetry pipelines for network-function evaluation.
