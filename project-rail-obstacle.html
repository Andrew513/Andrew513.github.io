Rail Track Obstacle Detection System

1st Place, International Innoserve Competition

YOLOUNetComputer VisionReal-time Pipeline

Overview

I built an end-to-end rail obstacle perception system that combines object detection and segmentation to improve decision confidence under real-world constraints.

What I Built

  • Integrated YOLO for detection and UNet for segmentation into a single perception workflow.
  • Designed post-processing logic to compare detection boxes and segmentation masks for consistency.
  • Implemented real-time notification and streaming components to emulate deployment environments.

Results

  • Improved detection accuracy by 30% through spatial-consistency analysis and false-positive reduction.
  • Improved robustness under lighting variation and occlusion by iterative model tuning.
  • Won 1st place in the International Innoserve Competition.
