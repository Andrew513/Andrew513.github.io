Overview

I implemented a compact relational database engine with core storage and query execution components under constrained memory budgets.

What I Built

  • Disk-backed page and record abstractions with LRU buffer pool management.
  • B+ tree indexing for efficient lookups and range-based access patterns.
  • External sorting and cost-aware join planning to support analytical query execution.

Outcome

  • Executed TPC-H scale-1 style analytical workloads under fixed memory limits.
  • Validated correctness and stability across query execution edge cases.
  • Strengthened practical understanding of storage, indexing, and optimizer tradeoffs.
