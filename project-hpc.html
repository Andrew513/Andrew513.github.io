GPU & High-Performance Computing

Rice University | CUDA, OpenMP, MPI, and performance analysis

CUDAOpenMPMPIHPCToolkit

Overview

This project explored parallel algorithm design and scalability behavior across multicore CPU and GPU environments, with emphasis on memory hierarchy effects and synchronization overhead.

What I Implemented

  • Dense linear algebra kernels in C++/OpenMP and CUDA for n=8000 class workloads.
  • CUDA kernels for flagging, prefix-sum scan, and scatter over 536M elements.
  • A 2.5D distributed matrix multiplication design using MPI 3D Cartesian process grids.

Performance Work

  • Profiled kernels with HPCToolkit to isolate synchronization and memory bottlenecks.
  • Analyzed scaling limits under bandwidth-bound and communication-heavy regimes.
  • Modeled compute vs. communication tradeoffs for architecture-aware optimization decisions.
