ShortKing URL Shortener

Rice University | Secure microservices on GCP

Java 18Spring BootMongoDB AtlasGCP Cloud Run

Overview

I designed and deployed a production-style URL shortener platform with secure authentication, link lifecycle controls, and analytics capabilities.

What I Built

  • Implemented REST APIs for shortening, redirecting, TTL expiration, and click analytics.
  • Built authentication with JWT, Spring Security, Google OAuth2 SSO, and SHA-256 token hashing flows.
  • Designed MongoDB aggregation pipelines for analytics dashboards and usage tracking.
  • Containerized and deployed services to Cloud Run with multi-region load balancing and CI/CD via Cloud Build.

Results

  • Achieved under 100ms average redirect latency.
  • Improved deployment reliability through repeatable cloud-native automation.
  • Established secure and maintainable architecture for extension into enterprise features.
